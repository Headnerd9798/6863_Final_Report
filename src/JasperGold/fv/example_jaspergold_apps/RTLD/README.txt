Example of how to start using JasperGold Apps for RTL Development
This example is split into 2 parts and with 2 language variations. 
You must run RTLD_start_<language>_example.tcl and generate a Jasper database
prior to running the RTLD_continue_<language>_example.tcl file.

To run: 
  jg -rtld RTLD_start_verilog_example.tcl RTLD_continue_verilog_example.tcl
or
  jg -rtld RTLD_start_vhdl_example.tcl RTLD_continue_vhdl_example.tcl
